Servidor FTP

Servidor FTP

Servidor FTP (File Transfer Protocol)

  • Serviço que permite o envio e recebimento de arquivos entre seu computador e o servidor onde seu site está hospedado
  • Ele facilita o upload, download e gerenciamento de arquivos do site
  • Para acessá-lo, você pode usar programas como FileZilla ou WinSCP

JavaScript

JS

JavaScript Programming Language

  • Uma linguagem de programação usada para tornar os sites interativos e dinâmicos
  • Com ele, você pode criar animações, validar formulários, exibir mensagens e modificar o conteúdo da página sem precisar recarregar
  • Ele roda no navegador do usuário e funciona junto com HTML (estrutura) e CSS (estilo) para criar experiências mais envolventes na web

Docker

Docker

  • Uma plataforma que permite criar, distribuir e executar aplicativos em containers
  • Um container é como um pacote que inclui tudo o que um software precisa para rodar, como código, dependências e configurações, garantindo que funcione da mesma forma em qualquer ambiente
  • Isso facilita o desenvolvimento, a implantação e a escalabilidade de aplicações, eliminando problemas de compatibilidade
